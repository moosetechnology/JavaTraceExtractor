# Downloading the Project
This section explains how to download the project, using Eclipse as a reference.

## 1. In Eclipse: Right-click in the empty area of the 'Project Explorer' and select 'Import...'

<img src="../image/download/download-1-Project-Explorer.png" width="70%">

## 2. Choose 'Projects from Git (with smart import)'

<img src="../image/download/download-2-project-from-git.png" width="70%">

## 3. Select the 'Clone URI' option

<img src="../image/download/download-3-clone-URI.png" width="70%">

## 4. In your web browser, go to the GitHub repository, click on '<> Code', and copy the SSH link

<img src="../image/download/download-4-github-link.png" width="70%">

## 5. Return to Eclipse and paste the link into the 'URI' field

<img src="../image/download/download-5-import-from-git.png" width="70%">

## 6. Click 'Next' and choose the branch to download (by default, import the 'main' branch)

<img src="../image/download/download-6-branch-selection.png" width="70%">

## 7. Choose the local destination directory and the project name

<img src="../image/download/download-7-Local-destination.png" width="70%">

## 8. Click 'Next', and then click 'Finish'

<img src="../image/download/download-8-Finish.png" width="70%">
